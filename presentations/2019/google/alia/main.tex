  Building and deploying a cooperatively governed labor market for domestic workers, with the \alert{NDWA}'s innovation arm, the \alert{Fair Care Labs}.

    \only<1>{
    \large{\textbf{What we did}}
    \begin{itemize}
      \item Built a system for domestic workers to manage appointments and correspondence with clients
      \item Interviewed, focus tested, and prototyped with nannies, caregivers, \& housecleaners in and around Oakland
    \end{itemize}
    }
    \only<2->{
    \large{\textbf{Some takeaways}}
      \begin{itemize}
        \item<0,2,4> \alert{Technical solutions} tend only to work for technical people
        \item<0,3,4> Workers need more \alert{privacy} than they're getting
      \end{itemize}
    }
